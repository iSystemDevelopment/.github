# AI Orchestration

Summary — full guide: [docs/AI_ORCHESTRATION.md](https://github.com/iSystemDevelopment/.github/blob/main/docs/AI_ORCHESTRATION.md)

---

## Principles

- **Human lead** owns scope, architecture, merge, and security sign-off.  
- **AI assistants** implement, review, and document under direction.  
- No secrets in chat or git.

---

## Session flow

1. Define outcome  
2. Plan (human confirms)  
3. Implement (small diffs)  
4. Verify (tests + visual QA for web)  
5. Document + PR  
6. Human merge  

---

## Web work

Must follow [isystem-deploy standards](https://github.com/iSystemDevelopment/isystem-deploy/tree/main/docs):

- Layout view modes  
- PWA + iOS  
- CSS `?v=` versioning + Cloudflare cache rules
